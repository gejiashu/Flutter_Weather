## 项目介绍

Flutter_Weahter是使用Flutter开发的一个简单的查询天气的app，功能比较简单

里面使用了ListView,GrideView，Column,Container等很多Flutter开发中常用的组件

同时也对app常见的网络请求，获取位置，本地存储等常见操作进行了简单的使用非常适合新手进行学习。

## 功能介绍

- 支持获取当前位置天气情况

- 支持手动选取其他城市

- 支持历史城市记录

- 支持查看未来五天的天气情况
